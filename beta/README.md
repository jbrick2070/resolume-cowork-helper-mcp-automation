# Beta

These files are useful experiments, not the verified stable release.

| Composition | Matching controller preset |
|---|---|
| `compositions/Res React Live Gen.avc` | `controllers/APC 40 MK II - React v4.4.xml` |
| `compositions/Res React Orbit Gen.avc` | `controllers/APC 40 MK II - Orbit v1.xml` |

Both compositions are generative and remain useful as references for the FFT
prompts, manifests, and FFGL plugins. Their APC40 controller pairings have not
been accepted as end-to-end verified builds. Use them for experimentation, not
as the stable install path.
