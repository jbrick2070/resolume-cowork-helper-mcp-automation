# Prompt 01 - First Contact (5 minutes, zero risk)

Copy everything below the line into your agent (Claude Code / Cowork with a
Resolume MCP connected). Have Resolume Avenue/Arena 7 running with any
composition open.

---

You are connected to my Resolume via an MCP server. This is a first-contact
exercise: READ a lot, WRITE almost nothing, so I can watch the loop work.

Ground rules for this and every future Resolume session:
- Never save, open, or create a composition without my explicit OK.
- Structural changes (layers/columns/decks) one at a time, and re-read state
  after each one before doing the next.
- Batch at most 8 operations per call.
- If a call times out, do NOT retry blind - read state back first.

Do this, in order:
1. Report server health and the composition name, layer count, column count.
2. List my layers with their blend modes and opacities.
3. List whatever is in column 1.
4. Take a monitor snapshot and describe what you actually see in pixels.
5. Rename exactly one thing: layer 1, to "React Test" (Title Case). Then
   read it back to prove it stuck, then rename it back to its original name.
6. Tell me one thing about my composition I probably did not know.

VERIFY: if every read matches what I see in the Resolume UI, first contact
worked. If anything disagrees, tell me which read lied - that matters.
