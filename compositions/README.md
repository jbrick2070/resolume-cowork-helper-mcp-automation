## APC40_Visual_QA_148.avc

This is the stable, verified composition: a 1920 x 1080 visual twin of the
APC40 mkII with 148 layers, one column, three decks, and 148 native Text
Animator clips. It pairs with:

    ../controllers/APC 40 MK II - Visual QA.xml

Install the preset, enable the APC40 mkII as both MIDI input and output, and
select `APC 40 MK II - Visual QA` in Preferences > MIDI. If labels are dark on
first open, trigger column 1 once.
